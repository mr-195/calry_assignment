
# Calry Coding Assignment



## Installation

Install with venv (python) in linux/windows

```bash
python3 -m venv venv
source venv/bin/activate
```
or in windows ...

```bash
.\venv\Scripts\activate
```
after this, do

```bash
pip install flask
python main.py
```
## Running Tests

To run tests, run the following command

```bash
  chmod +x script.sh
  ./script.sh
```

-Ensure flask is installed
